## Get started

```sh
npm install @martendebruijn/css
```

## Usage

Add `style.css` to you html file:

```html
<link
  rel="stylesheet"
  href="../node_modules/@martendebruijn/css/dist/styles.css"
/>
```

Change the defaults:

```css
:root {
  --body-color: green;
  --body-bg: hotpink;
}
```
